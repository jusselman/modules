What the F is a module?

Modules are used in javascript in order to maintain certain pieces of code independent from each other. In this page's javascript we see three modules: 1. Budget Controller 2. UI Controller 3. Controller. The Controller module is necessary to make the other two communicate, thus if we wanted to change either of the two former, we will still be able to use whatever we replace them with. This can be seen in the JS file. In the Console, type in budgetController.publicTest(), and fill it with whatever parameter you want. you'll notice that this will utilize closure in over to capture the information saved in the variables in Budget Controller which would otherwise be innacessible.
